Introdução ao Microsoft Word

O Microsoft Word é um editor de texto que permite criar documentos profissionais com formatação avançada.

Interface do Word

  • Faixa de Opções - Barra superior com ferramentas
  • Área de Trabalho - Onde você digita o texto
  • Barra de Status - Informações do documento
  • Painel de Navegação - Para navegar pelo documento

Formatação Básica

  • Negrito - Ctrl + B
  • Itálico - Ctrl + I
  • Sublinhado - Ctrl + U
  • Tamanho da fonte - Selecionar na barra
  • Cor do texto - Botão A colorido

Exercício Prático

  1. Abrir o Word
  2. Digitar seu nome em negrito
  3. Escrever um parágrafo sobre você
  4. Criar uma lista de hobbies
  5. Salvar o documento

Dicas Importantes

  • Salve o documento frequentemente (Ctrl + S)
  • Use estilos para manter consistência
  • Revise a ortografia (F7)
  • Faça backup dos documentos importantes
