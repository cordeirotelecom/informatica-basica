Navegando na Internet

A internet é uma rede mundial de computadores conectados que permite compartilhar informações.

Principais Navegadores

  • Google Chrome - Mais popular mundialmente
  • Mozilla Firefox - Código aberto e privacidade
  • Microsoft Edge - Integrado ao Windows
  • Safari - Padrão no Mac

Conceitos Importantes

  • URL - Endereço de um site (www.exemplo.com)
  • HTTP/HTTPS - Protocolos de comunicação
  • Cookies - Arquivos temporários dos sites
  • Cache - Memória temporária do navegador

Segurança Online

  • Verificar se o site é seguro (https://)
  • Não compartilhar senhas
  • Cuidado com links suspeitos
  • Manter o navegador atualizado
