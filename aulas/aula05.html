Introdução ao Microsoft Excel

O Excel é uma planilha eletrônica usada para cálculos, organização de dados e criação de gráficos.

Conceitos Básicos

  • Células - Interseção de linha e coluna (A1, B2, etc.)
  • Planilha - Conjunto de células organizadas
  • Pasta de Trabalho - Arquivo que contém planilhas
  • Fórmulas - Começam sempre com = (igual)

Fórmulas Básicas

  • =SOMA(A1:A10) - Soma valores de A1 até A10
  • =MÉDIA(B1:B5) - Calcula a média
  • =MÁXIMO(C1:C20) - Encontra o maior valor
  • =MÍNIMO(D1:D15) - Encontra o menor valor
  • =CONTAR(E1:E10) - Conta células com números

Exercício: Controle de Gastos

  1. Criar planilha com colunas: Data, Descrição, Valor
  2. Inserir 10 gastos fictícios
  3. Calcular o total de gastos
  4. Encontrar o maior e menor gasto
  5. Calcular a média de gastos

Formatação

  • Formato de moeda (R$)
  • Formato de data
  • Cores de células
  • Bordas e alinhamento
